# An English vocabulary web crawler and memorization assistant system

## 為什麼做這個專案?
1.只針對大考中心發布的單字表中的單字與其詞性爬取資料
108課綱以來出現過的難題其實都在大考中心單字表範圍內
2.程式自動安排府和記憶曲線的複習時間

3.詞卡系統讓使用者著重在英文句子而非單字


為了有效應付學測會考但訪間參考書沒有的內容
