# An English vocabulary web crawler and memorization assistant system

## why this project?
1.針對大考中心發布的單字表中的單字與其詞性爬取資料

2結合上述功能的詞卡記憶輔助功能

## how to use?
### 首先是爬蟲的部分
打開電腦的終端或powershell，並把以下內容複製貼上
 ```
pip install requests
```
 ```
pip install BeautifulSoup
```
